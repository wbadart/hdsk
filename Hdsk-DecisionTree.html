infoGain :: (Foldable f, Alternative f, Eq label, Ord label) => (tup -> label) -> f tup -> [tup -> Bool] -> Double Source #

O(nC + n log n) Compute the information gain of the given branching. Branches are encoded as a list of predicates over data instances. An object belongs to whichever branch for which it passes the predicate. Assumes that each data object will pass one and only one of the branching predicates.
