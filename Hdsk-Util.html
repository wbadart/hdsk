majorityLabel :: (Foldable f, Functor f, Eq label, Ord label) => (tup -> label) -> f tup -> label Source #

O(n log n) Compute the most frequent label in the data set. Ties do not have well defined behavior; the tie is broken by maximumBy which doesn't have clear documentation on ties.

count :: (Foldable f, Ord k) => f k -> Map k Int Source #

O(n log n) Given a container, construct a map from the unique elements of the container to their frequency within the container.

countBy :: Foldable f => (a -> Bool) -> f a -> Int Source #

O(n) Given a container and a predicate, calculate the number of elements which pass the predicate

countBy' :: (Foldable f, Num b) => (a -> Bool) -> f a -> b Source #

O(n) Version of countBy with generic result.

countHT :: (Foldable f, Eq k, Hashable k) => f k -> ST s (HashTable s k Int) Source #

O(n) Given a container, construct a hash table from the unique elements of the container to their frequency within the container. I recommend this function over count when 1) counting performance is critical, AND 2) you're familiar with the ST monad.

countElemHT :: (Foldable f, Eq k, Hashable k) => f k -> k -> Int Source #

O(n) Convenience function for getting the count of a single element from a container.

length' :: (Num a, Foldable f) => f b -> a Source #

O(n) Shorthand for genericLength which works on all foldables.

uniq :: Ord a => [a] -> [a] Source #

O(n log n) Filter the duplicates out of a list.

uniq' :: (Foldable f, Alternative f, Ord a) => f a -> f a Source #

O(n log n + nX) where X is the const of the associative binary operator for the given Alternative type. Filter the duplicates out of a Foldable, Alternative container.
