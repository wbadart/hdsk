O(n) Computes the unbiased variance of a collection of numbers.

std :: (Foldable f, Functor f, Floating n) => f n -> n Source #

O(n) Computes the standard deviation of a collection of numbers.

percentile :: (Selectable p, RealFrac n) => n -> p n -> n Source #

O(n) Selects the element which is greater than p% of the rest. When the p-th percentile does not land directly on a whole index, midpoint interpolation is used to average left and right side of the split.

median :: (Selectable p, RealFrac n) => p n -> n Source #

iqr :: (Selectable p, RealFrac n) => p n -> n Source #

O(n) Inter-quartile range. The distance between the first and third quartiles.

select :: (Selectable p, Ord a) => Int -> p a -> a Source #

O(n) Simple implementation of quickselct (aka Hoare's algorithm or k-rank). Selects the k-smallest element from the collection.
