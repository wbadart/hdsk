Hdsk.NearestNeighbors

Description

This module implements the K Nearest Neighbors classifier, which finds the k tuples "closest" to the unobserved instance and combines their labels via some voting/ consensus metric to produce a prediction.

knn :: (tup -> tup -> Double) -> Int -> [tup] -> tup -> [tup] Source #

O(nD log k) where D is the cost of the distance function. Select the k nearest training instances to the argument tuple, according to the given distance metric (greater value means more dissimilar).
